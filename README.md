# wordle-helper
Simple TUI to cheat on wordle

https://github.com/user-attachments/assets/2d5aa59e-5bad-45f6-979d-c4dea76516a4


## Usage

| Keymap             | Action           |
| ------------------ | ---------------- |
| `Shift` + `letter` | Yellow colouring |
| `Alt` + `letter`   | Green colouring  |
| `Return`           | Submit word      |
| `Ctrl` + `c`       | Exit             |
